# Kohrus Landing Page — Card Image Style Only

This applies **only to images inside cards** on the landing page (hero card + 4 featured asset cards). Rest of the UI is unchanged.

## Image palette (inside the artwork only)

- **Warm paper/cream:** #F5F0E8, #EDE6DC
- **Terracotta / burnt earth:** #C17F59, #A65D3A, #8B4513
- **Ochre / warm gold:** #C9A227, #D4A84B
- **Deep indigo / ink:** #2C3E5D, #3D4F6B
- **Warm neutrals:** #6B6360, #4A4543

## Image style (authentic, handmade, unique)

- **Look:** Linocut or soft paper-collage feel; subtle texture; not glossy or tech.
- **Music theme:** Abstract (instruments, waveforms, vinyl, notation) — artful, not literal.
- **No:** Generic purple/blue gradients, cold fintech look, stock photos.

## Usage

- **Hero card:** One image (`hero-card.png`), ~384×192px area (aspect ~2:1), “featured music asset” mood.
- **Featured asset cards:** Four images (`featured-1.png` … `featured-4.png`), same palette and style.
- **How It Works steps:** Three images (`how-step-1.png`, `how-step-2.png`, `how-step-3.png`), same thematic style.

Images are generated by `scripts/edit_landing_images.py` (Nano Banana image-to-image from Unsplash sources, or text-to-image for section art) and saved to `public/`.

- **Hero:** `hero-card.png` (card), optional `hero-bg.png` (subtle left-side texture).
- **Featured:** `featured-1.png` … `featured-4.png`.
- **How it works:** `how-step-1.png` … `how-step-3.png`.
- **Why Kohrus:** `why-1.png`, `why-2.png`, `why-3.png` (fractional, passive, liquid).
- **Who it's for:** `for-investors.png`, `for-artists.png`.
- **CTA:** `cta-banner.png`.

Env flags: `ONLY_HOW_STEPS=1`, `ONLY_CTA_BANNER=1`, `ONLY_NEW_SECTIONS=1` (hero-bg + why 1–3 + for-investors + for-artists).
